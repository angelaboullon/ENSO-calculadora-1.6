# Desktop Calculator in Java/Swing

- **Maven** project
- GitHub Actions CI workflow for automated building and testing. 
- Scrum-style issue templates and labels.

## Prerequisites
- Java Development Kit (JDK) 21
- Apache Maven 3.9+

## Project Structure
- `src/main/java`: Source code.
- `src/main/resources`: Icons
- `src/test/java`: Test code.
- `pom.xml`: Maven configuration and dependencies.

## Build
```
mvn clean package
```

## Run
```
java -jar target/enso-calculadora-0.0.1-SNAPSHOT.jar
```

## Test
```
mvc test
```
